# Fix Plan Template

Use this template when presenting the fix proposal in Phase 4.

```
## Fix Plan: {issue title} (#{number})

### Root Cause
[What is actually wrong and why it happens]

### Proposed Fix
[Specific changes to make, file by file]

### Files to Modify
| File | Change |
|------|--------|
| path/to/file.py | Description of change |

### Files to Create (if any)
| File | Purpose |
|------|---------|
| path/to/new_file.py | Why this file is needed |

### Test Strategy
- [ ] [Test case 1 — what it verifies]
- [ ] [Test case 2 — regression guard]

### Risks
- [Anything that could go wrong or affect other features]
```
